New Personal Website is Live

2023-01-29 :: Jason Dyke
#  # 
Over my parental leave I migrated my personal website from Wix to a Hugo site hosted on GitHub pages. Wix ended up costing too much for the value it provided, but I’m sure it’s great for professional companies who need a no code solution. This site runs entirely on GitHub Pages which offers free hosting, automatic updates via GitHub actions, generation and rotation of SSL certificates, and best of all gives me full control over the layout and content of my own personal website.
